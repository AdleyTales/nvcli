# nvcli

![npm version](https://img.shields.io/npm/v/nvcli.svg)
![npm downloads](https://img.shields.io/npm/dt/nvcli.svg)

> node cli toolkit.

## Install

```
npm install -g nvcli
```

## Usage

### 显示本地IP 

```
nvcli -i
```

### 压缩文件 

```
nvcli -g input.txt,input.txt.gz
```

### 网址转二维码

```
nvcli -q http://bestvist.com
```

### 浏览器打开网址

```
nvcli -o http://bestvist.com
```
